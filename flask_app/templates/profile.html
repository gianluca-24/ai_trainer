🎤 Voice Commands

During workout configuration:

repetitions

Set the number of repetitions for each set (1-100). Example: "six repetitions" or "ten repetitions"

sets

Set the number of sets for your workout (1-10). Example: "sets three" or "five sets"

timer

Set the rest time between sets (60-300 seconds). Example: "timer ninety" or "60 seconds timer"

start

Begin the workout with the current configuration

cancel

Cancel the workout setup and return to profile

During rest periods:

good

Keep the current workout settings for the next set

easy

Add one more set to your workout if it feels too easy

difficult

Reduce the number of repetitions by 2 for remaining sets if too challenging

terminate

End the workout session and save your progress
